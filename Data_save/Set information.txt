---------------------------- Information sur les sets de données -------------------------
Les Oréos sont répartis dans 4 groupes différents, pour pouvoir distinguer les sets de test, d'apprentissage, ... et éventuellement pouvoir les faire varier.
Ils sont donc distingués selon 4 couleurs : Rouge, Vert, Bleu, Noir. (Marqués sur la tranche)


---------------------------- Les différentes classes -------------------------
Les différentes classes sont :
- Les Oréos sans défauts
- Les Oréos fissurés
- Les Oréos effacés (= Mauvaise impression du dessin)
- Les Oréos mangés (= Avec une trace de dents sur un bord)
- Les Oréos décalés (= La crème dépasse du gâteau)
- Les gâteaux "non-oréo" (ex : pepito)

Certains Oréos du set peuvent comporter plusieurs défauts à la fois. Cependant, si un Oréo a une classe de défaut sur une face, il l'a forcément sur l'autre également
